eliminar_por_id()

eliminar_por_id(integer  $idActividad, boolean  $eliminar_imagen = true) : boolean

Función eliminar_por_id del modelo Actividad_model.

Esta función se encarga de eliminar una actividad dado su id. La función recibe un parámetro para indicar si sebe eliminar la imagen asociada a la actividad del servidor o si solo debe ser eliminada de la base de datos.

Parameters

integer $idActividad

Identificación única de la actividad.
boolean $eliminar_imagen

Valor booleano para indicar si se debe intentar eliminar la imagen asociada a la actividad o solo eliminar la actividad de la base de datos.

Returns

boolean —

Retorna true si se pudo eliminar, sino retorna false.
