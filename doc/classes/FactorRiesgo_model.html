eliminar_por_id()

eliminar_por_id(integer  $idFactorRiesgo, boolean  $eliminar_imagen = true) : boolean

Función eliminar_por_id del modelo FactorRiesgo_model.

Esta función se encarga de eliminar un factor de riesgo dado su id. La función recibe un parámetro para indicar si sebe eliminar la imagen asociada al factor de riesgo del servidor o si solo debe ser eliminada de la base de datos.

Parameters

integer $idFactorRiesgo

Identificación única del factor de riesgo.
boolean $eliminar_imagen

Valor booleano para indicar si se debe intentar eliminar la imagen asociada al factor de riesgo o solo eliminar el factor de riesgo de la base de datos.

Returns

boolean —

Retorna true si se pudo eliminar, sino retorna false.
